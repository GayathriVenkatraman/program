# Cloud & AI

This specialization course focuses on cloud computing and artificial intelligence using Amazon Web Services (AWS), preparing students for modern cloud-native development and AI-powered applications.

## Learning Goals

By the end of this course, students will be able to:

- AWS cloud fundamentals and services
- Deploy and manage applications in cloud environments
- Work with AI/ML services and APIs
- Build cloud-native applications
- Implement AI-powered features in web applications
